Approach :

First of all, let us state what the increment operators are and what do they do?

  • If the ++ i.e. the increment operator is written after the variable like i + +, then it acts as a post increment operator. This means that if we write x = i++ then the value of x will be i and not i + 1 because the post increment operator returns the old value and then increments the value.
  • If the ++ i.e. the increment operator is written before the variable like ++j then it acts as a pre increment operator i.e. if we write y = ++j, the value of y will be j + 1 as first the increment takes place and then the assignment.

From this, we can conclude that the post increment operator returns the old value whereas the pre increment operator returns the new value.
