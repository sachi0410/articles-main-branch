Understanding problem:

In this problem you are given a number n.
All you need to do is to calculate the value of 7n/8 without using division and multiplication.

For example:
Sample Input: 15
Sample Output: 13

How?
Well the answer would be = (7*15)/8 = 13.125
Since we are dealing in integers, we neglect the part after decimal, which gives us 13.

Approach :
Moving On:
  1. The problem here deals with solving a particular expression of the type 7*n/8 using bit manipulation only i.e. we are not allowed to use multiplication and division.
  2. Mathematical analysis:
    => 7*N/8 = (8*N - N)/8
    => (8*N - N)/8 = (((N << 3) - N) >> 3)
  3. Here (N << 3) is a left shift to 3 positions and as we know that a left shift of bits is the same as multiplication of a decimal integer by 2 so shifting by 3 positions is the same as multiplying a number by 8.

  5. After that we simply subtract N from this result which will result in 7*N in our numerator.
  6. As we know that right shift by a position is equivalent to division by 2 in the decimal number system so we need to divide the numerator by 8 so we will be shifting the numerator to right by 3 positions.
